# Figma Make Prompt — KUPC (Kathmandu University Placement Connect)

---

**Prompt:**

Design a complete, modern, premium web application UI for **KUPC (Kathmandu University Placement Connect)**, an exclusive Tinder-style recruitment platform connecting Kathmandu University students with verified employers.

The design should feel like a combination of **LinkedIn, Tinder, Notion, Stripe Dashboard, and Linear**, using clean layouts, generous whitespace, rounded corners, subtle shadows, smooth animations, and a professional SaaS aesthetic.

The application should be desktop-first with responsive behavior for tablets and mobile.

# Design Language

Create a premium SaaS interface.

Visual style should be:

* Minimal
* Modern
* Corporate
* Trustworthy
* Friendly
* High-end
* Professional
* Fast
* Clean

Avoid:

* Heavy gradients
* Glassmorphism
* Skeuomorphism
* Busy layouts
* Thick borders

Use:

* Rounded cards (16px radius)
* Soft shadows
* Large spacing
* Floating panels
* Smooth hover states
* Micro interactions
* Elegant typography

# Accessibility

Ensure WCAG AA contrast compliance, keyboard navigation, visible focus states, semantic color usage (don't rely on color alone), and scalable typography. Include both light and dark mode variants.

---

# Deliverables

Produce a cohesive, production-ready UI kit and high-fidelity mockups for all screens, using reusable Auto Layout components, variables, color styles, text styles, and component variants. Maintain consistent spacing, alignment, and visual hierarchy across the entire application. The final result should look polished enough for a modern startup MVP and be ready for handoff to frontend developers using React/Next.js and Tailwind CSS.
